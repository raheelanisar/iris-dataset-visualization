## Objective
Explore and visualize the famous Iris dataset to understand 
the distribution and relationships between different flower 
species using Python.

## Dataset
- **Name:** Iris Dataset
- **Source:** Kaggle
- **Rows:** 150
- **Species:** Iris-setosa, Iris-versicolor, Iris-virginica

## Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

## Visualizations
- 📊 Scatter Plot — Sepal Length vs Sepal Width
- 📊 Scatter Plot — Petal Length vs Petal Width
- 📦 Box Plot — Feature Distribution by Species
- 🔗 Pair Plot — All Features Comparison

## Model Used
- **Random Forest Classifier**
- **Accuracy: 97.78%**

## Key Insights
- Setosa species is easily separable from other species
- Petal Length and Petal Width are most important features
- Model achieved 97.78% accuracy on test data
